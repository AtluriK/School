Complete Introduction to Python

1. What is Python?

Python is a popular, easy-to-learn programming language used for web development, data analysis, automation, AI, and more. It emphasizes readable code and simplicity.

print("Hello, Python!")

Output:

Hello, Python!

2. How Python Fits Into Programming

Python is a general-purpose language used for:

  • Web and software development
  • Data science and machine learning
  • Scripting and automation
  • Game development

3. Why Learn Python?

  • Simple and readable syntax.
  • Large community and lots of libraries.
  • High demand in the job market.
  • Versatile for many types of projects.

4. Python Syntax Basics

Python uses indentation to group code blocks instead of braces.

name = "Alice" print("Hello, " + name)

Output:

Hello, Alice

5. Variables and Data Types

Variables store data and do not need explicit declaration.

  • str: Text data, e.g., "hello"
  • int: Whole numbers, e.g., 10
  • float: Decimal numbers, e.g., 3.14
  • bool: True or False
  • list: Ordered collections
  • dict: Key-value pairs
age = 25 height = 5.7 is_student = True fruits = ["apple", "banana", "cherry"] person = {"name": "Bob", "age": 30} print(age, height, is_student) print(fruits[1]) print(person["name"])

Output:

25 5.7 True
banana
Bob

6. Operators

Operators perform calculations or comparisons.

x = 10 y = 3 print(x + y) # Addition print(x > y) # Comparison

Output:

13
True

7. Control Flow: if, else, elif

Run different code based on conditions.

age = 18 if age >= 18: print("Adult") elif age > 12: print("Teenager") else: print("Child")

Output:

Adult

8. Loops: for and while

Repeat code multiple times.

For loop example:

for i in range(1, 6): print("Count:", i)

Output:

Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

While loop example:

count = 1 while count <= 5: print(count) count += 1

Output:

1
2
3
4
5

9. Functions

Functions group reusable code blocks.

def greet(name): print("Hello, " + name + "!") greet("Bob")

Output:

Hello, Bob!

10. Lists and Dictionaries

Lists are ordered collections. Dictionaries store key-value pairs.

fruits = ["apple", "banana", "cherry"] print(fruits[0]) person = {"name": "Alice", "age": 25} print(person["age"])

Output:

apple
25

11. Error Handling

Use try...except to handle errors and prevent program crashes.

try: print(10 / 0) except ZeroDivisionError: print("Cannot divide by zero!")

Output:

Cannot divide by zero!

12. Modules and Libraries

Modules are files containing Python code. Libraries are collections of modules for extra functionality.

Example: Using the math module.

import math print(math.sqrt(16))

Output:

4.0

13. File Handling Basics

Python lets you read and write files easily.

with open("example.txt", "w") as file: file.write("Hello, file!") with open("example.txt", "r") as file: content = file.read() print(content)

Output:

Hello, file!
